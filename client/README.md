InterviewIQ 🚀

Prepare Smarter. Perform Better.

InterviewIQ is a full-stack MERN SaaS application that helps users prepare for interviews using AI-powered features. Users can upload resumes, generate interview questions, practice technical & HR rounds, and receive intelligent feedback — all powered by a scalable credit-based system. This project focuses on real-world SaaS architecture, payments integration, authentication, and production deployment rather than basic CRUD functionality.

✨ Core Features

✅ Upload Resume (PDF) ✅ AI-Generated Interview Questions ✅ Technical & HR Practice Rounds ✅ Intelligent Feedback System ✅ Credit-Based Access Control ✅ Secure Payments via Razorpay ✅ Firebase Google Authentication ✅ Smooth UI with Framer Motion ✅ Full Stack Deployment on Render

🧠 Project Vision InterviewIQ simulates a modern SaaS product where:

• AI enhances user experience • Access is monetized via credits • Payments & authentication mirror production apps • Backend structure follows scalable practices

💻 Tech Stack

Frontend • React.js • Framer Motion • Axios

Backend • Node.js • Express.js • MongoDB • Mongoose

Authentication • Firebase Google Authentication

Payments • Razorpay Integration

Deployment • Render (Frontend + Backend)

🔐 Authentication Flow Users sign in securely using Firebase Google Authentication. No traditional username/password storage required.

💳 Credit System InterviewIQ uses a credit-based access model:

• Users consume credits for AI features • Credits can be purchased via Razorpay • Backend enforces feature access based on balance

This mirrors real SaaS monetization logic.

💰 Payments Integration Secure payments are handled using Razorpay:

• Create orders via backend • Complete transactions on frontend • Verify payments server-side

🚀 Deployment The application is deployed using Render, simulating a production-ready full stack environment.

🔥 Why This Project Matters This is NOT a basic CRUD app.
